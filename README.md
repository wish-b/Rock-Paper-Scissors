🎮 Rock Paper Scissors (Python)

A simple command-line Rock Paper Scissors game built using Python.
This project is part of my learning journey and shows how I improved my coding logic step by step.

---

🚀 Features

- Play against the computer 🤖
- Random computer choices each round
- User input handling (Rock, Paper, Scissor)

---

🛠️ Technologies Used

- Python 3
- Built-in "random" module

---

📂 How to Run

1. Clone this repository:

git clone https://github.com/wish-b/Rock-Paper-Scissors.git

2. Navigate to the project folder:

cd rock-paper-scissors

3. Run the program:

python main.py

---

🎯 How to Play

- Enter your name
- Choose:
  - "r" for Rock
  - "p" for Paper
  - "s" for Scissor
- The computer will randomly choose its move
- The winner will be displayed
- And to play again run the program
